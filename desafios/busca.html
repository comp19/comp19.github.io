# Algoritmos de Busca

# Exercício 1 - Nomes

Construa um array de nomes a partir dos inputs do utilizador.

Exemplo:

String[] names = {"João", "Leonor", "Fernando", "Joana", "Pedro", "Maria"};

Em seguida, imprima todos os nomes.

# Exercício 2 - Valor máximo

Procura no array integerList o maior valor.

int[] integerList = {52, 81, 33, 17, 25, 90, 28, 43};

Imprima o valor encontrado.

# Exercício 3 - Busca pelo índice

Procura no array integerList um número inteiro inserido pelo utilizador. Caso o número esteja no array, imprima a posição (índice). Se não, imprima um feedback adequado ao utilizador.

int[] integerList = {52, 81, 33, 17, 25, 90, 28, 43};

# Exercício 4 - Busca (rápida) pelo índice

Procura no array orderedIntList um número inteiro inserido pelo utilizador. Caso o número esteja no array, imprima a posição (índice). Se não, imprima um feedback adequado ao utilizador.
Repara agora que o array está ordenado de menor para maior. Qual algoritmo podemos implementar para tornar a procura mais eficiente?

int[] orderedIntList = {1, 2, 11, 12, 15, 17, 25, 26, 28, 28, 29, 33, 43, 44, 47, 50, 52, 61, 73, 81, 82, 90, 99, 100};
