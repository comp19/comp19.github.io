# FizzBuzz Dois

# Objetivo

A partir do programa abaixo (que resolve a tarefa incial), considere como podemos melhorá-lo.
A proposta é desenvolver a habilidade de reconhecer padrões. Qual instrução é repetida desnecessariamente e podemos assim generalizar?

import java.util.Scanner;

public class FizzBuzz {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);
        int userInput;

        do {
            System.out.println("Introduza um número inteiro positivo");
            userInput = scanner.nextInt();
        } while (userInput < 0);

        scanner.close();

        for(int i = 1; i <= userInput; i++) {

            if (i % 3 == 0 && i % 5 == 0) {
                System.out.println("FizzBuzz");

            } else if (i % 3 == 0) {
                System.out.println("Fizz");

            } else if (i % 5 == 0) {
                System.out.println("Buzz");

            } else {
                System.out.println(i);
            }
        }
    }
}

Por exemplo, após cada condicional repetimos a instrução de “println()” onde o que muda é a mensagem a ser impressa. Em outras palavras, o que realmente está sujeito a uma mudança de acordo com a condicional, é apenas a mensagem e não a instrução de “println()”.

Algumas dicas:

  • Utilizar uma variável message do tipo String.
  • Apenas um “println()” ao final do programa.
  • Não utilizar else ou else if.
